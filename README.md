
# Setup Guide

Berikut cara install / setup project

sebelumnya install dulu python 3.8
url: https://www.python.org/ftp/python/3.8.0/python-3.8.0-amd64.exe



## Deployment

setup environment
```bash
  virtualenv nama_env
```


lalu aktifkan env nya
```bash
  nama_env\Scripts\activate
```


setelah env aktif, lanjut install dependency
```bash
   pip install -r requirements.txt
```

jalankan aplikasi
```bash
  python app.py
```

Lalu buka http://localhost:5000/
